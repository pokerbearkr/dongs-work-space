데이터 블록

데이터베이스 I/O의 단위
논리저장구조의 최소 단위
운영체제 블록과 크기가 다를 경우 배수 단위로 데이터 블록의 크기를 설정


행 연쇄(Row Chaining) & 행 이주(Row Migration)
Insert 수행시 공간부족으로 데이터가 모두 저장되지 않는경우 데이터를 잘라서 다른곳에 넣어두고 포인터를 남겨둔다.
Update 수행시 큰 값으로 바뀌면서 공간이 없는경우 통째로 옮겨서 다른곳에 두고 포인터만 남겨둠

The Parameter File
Instance 시작할때 Parameter file($TB_SID.tip)을 read
Control file의 위치와 이름이 들어있음
txt파일임 

slog(System log)
예전에는 TRACE log 라고 불렸음
Error 발생시 slog or Dbms log에 적음
성능이 저하되는 원인을 찾거나 티베로 자체버그를 해결하는데 사용함

dlog(DBMS log)
서버 기동 및 종류,DDL문장의 수행등이 기록되는 파일
DDL,Server Manager Statements(STARTUP,SHUTDOWN,ARCHIVE LOG and RECOVER)위 항복 에러 발생시 적음

티베로 기능

TAC(Tibero Active Cluster)
공유디스크 기반의 Active Clustering을 통하여 다양한 유형의 장애에도 시스템 중단없이 서비스 지원

TSC(Tibero Standby Cluster)
독립 디스크 방식의 TSC이중화 구성을 지원하여 고가용성과 재해복구에 적합한 아키텍처를 제공함

보안 및 암호화
TDE 컬럼 암호화, TDE 테이블스페이스 암호화 지원

개발관리도구
dbAdmin/tbStudio, T-up tbExport/tbImport, tbLoader, tbrmgr, tbSQL 등등 다양한 Utility를 지원
